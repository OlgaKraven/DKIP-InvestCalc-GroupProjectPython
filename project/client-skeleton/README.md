## Client Skeleton — минимальный клиент для взаимодействия с API

Этот каркас предназначен для быстрого старта фронтенд-клиента,
который общается с FastAPI-приложением через REST API.

## Запуск
Просто откройте файл public/index.html в браузере.

## Функциональность
- Просмотр списка items
- Добавление нового item через API

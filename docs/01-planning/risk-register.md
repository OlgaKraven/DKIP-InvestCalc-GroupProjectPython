## Методика оценки рисков

- **Вероятность:** низкая / средняя / высокая  
- **Влияние:** низкое / среднее / высокое  
- **Уровень риска:**  
  - *низкий*: принимается  
  - *средний*: контролируется  
  - *высокий*: требует мер  
  - *критический*: угрожает проекту

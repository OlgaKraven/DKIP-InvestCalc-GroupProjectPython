## 3. Реализация backend (API + бизнес-логика) (Занятия 3–4)

### Цели

* Сделать рабочее FastAPI/ASP.NET API.
* Реализовать расчёт TCO/ROI/Payback и анализ чувствительности.
* Организовать хранение сценариев в JSON-файлах.

### Что делаем

* [ ] `src/main.py`

  * конфигурация FastAPI,
  * маршруты,
  * `/health`, `/docs`.

* [ ] `src/models/invest.py`

  * `InvestInput`, `InvestResult`,
  * `SensitivityRequest`, `SensitivityResult`,
  * `ScenarioShort`, `ScenarioDetail`.

* [ ] `src/services/invest_service.py`

  * функции расчёта,
  * анализ чувствительности ±N%,
  * работа с JSON-хранилищем.

* [ ] `src/api/v1/routes_invest.py`

  * `POST /api/v1/calc`,
  * `POST /api/v1/sensitivity`,
  * `GET/POST /api/v1/scenarios`.

### Артефакты

* Рабочее API (Swagger).
* Обновление справочника API (`docs/05-api/api-reference.md`).

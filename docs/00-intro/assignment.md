# Задание на групповой проект - InvestCalc — Инвестиционный аналитик ИС

Данный документ содержит официальное задание для выполнения группового проекта в рамках учебной практики / междисциплинарного модуля ПИДИС. Проект выполняется командой из 2–4 человек и включает полный цикл разработки мини-информационной системы.

---

## 1. Цель проекта

Разработать учебную информационную систему **InvestCalc**, предназначенную для расчёта экономической эффективности внедрения информационных систем (ИС), включая:

- расчёт **TCO** (Total Cost of Ownership),
- расчёт **ROI** (Return on Investment),
- расчёт **Payback Period** (срок окупаемости),
- **анализ чувствительности ±20%**,
- сравнение альтернативных сценариев (локальная vs облачная инфраструктура).

Проект должен включать:
- документацию,
- программную реализацию,
- тестирование,
- DevOps-сопровождение,
- итоговый отчёт и презентацию.

## 8. Формат работы в команде

### 8.1. Состав команды: 2–4 человека
Каждый участник выполняет **сопоставимый объём**:
- документация,
- код,
- DevOps,
- тестирование.

### 8.2. Требования к GitHub
- репозиторий должен быть публичным,
- используется GitHub Flow (main + feature-*),
- минимально 3 pull-request на участника,
- минимально 10 коммитов на участника.

### 8.3. Checker
- преподаватель проверяет:
  - структуру репозитория,
  - документацию,
  - корректность расчётов,
  - запуск API,
  - Docker,
  - CI/CD,
  - отчёт.

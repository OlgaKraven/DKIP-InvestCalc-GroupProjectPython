# Presentation Outline — Структура и сценарий выступления

Документ помогает участникам команды правильно распределить роли и выступать слаженно.

---

## 1. Роли докладчиков

Рекомендуемая модель:

- **Докладчик 1** — постановка задачи, предметная область  
- **Докладчик 2** — архитектура, реализация  
- **Докладчик 3** — расчёты, демонстрация  
- **Докладчик 4** — DevOps, выводы

## 3. Советы по представлению

- тренируйтесь выступать по очереди;
- не читайте текст с экрана — объясняйте;
- показывайте только ключевые точки, не загружая формулами;
- оставляйте время на вопросы.

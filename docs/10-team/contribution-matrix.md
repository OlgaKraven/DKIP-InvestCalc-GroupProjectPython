## Contribution Matrix — Матрица вклада участников проекта

Матрица показывает вклад каждого участника команды в различные части проекта.
Используется для объективной оценки командной работы.

---

## 1. Формат матрицы

| Раздел проекта | Участник 1 | Участник 2 | Участник 3 | Участник 4 |
|----------------|------------|------------|------------|------------|
| **Аналитика и требования** |  |  |  |  |
| **Архитектура (C4, ADR)** |  |  |  |  |
| **Backend (FastAPI)** |  |  |  |  |
| **Бизнес-логика расчётов** |  |  |  |  |
| **Анализ чувствительности** |  |  |  |  |
| **Тестирование (API+unit)** |  |  |  |  |
| **DevOps (Docker, CI/CD)** |  |  |  |  |
| **Документация** |  |  |  |  |
| **Презентация** |  |  |  |  |
| **Отчёт проекта** |  |  |  |  |

---

## 2. Легенда

- **1** — минимальный вклад  
- **2** — умеренный вклад  
- **3** — значительный вклад  
- **4** — лидер направления  

---
